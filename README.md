## Proje 1: Çift Katmanlı Web Uygulaması (Web API + Frontend)

Bu proje kapsamında, RESTful API mimarisine dayalı çift katmanlı bir web
uygulaması geliştirilmiş ve Amazon Web Services (AWS) bulut altyapısı üzerinde
canlı ortama taşınmıştır. Sistem; kullanıcı arayüzü (frontend), iş mantığı ve API
(backend) ile veritabanı (database) katmanlarının birbirinden bağımsız ve entegre
çalıştığı bir bulut mimarisi olarak tasarlanmıştır.
